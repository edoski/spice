Spice beginner guide

Objectives

Objectives tell training and tuning which metric to optimize and whether bigger or smaller is better. They can point at validation metrics or evaluator-backed economic metrics.

src/spice/objectives Docs: ARCHITECTURE.md, IMPLEMENTATIONS.md

Where It Fits

They sit between config/modeling/evaluation. Modeling compiles an objective runtime from objective config and metric descriptors.

Beginner Mental Model

A model can produce many scores. The objective is the scoreboard rule that says which score decides "best".

Main Source Files

base.py
Defines objective direction, objective config, compiled objective contracts, config coercion, and compile logic.
__init__.py
Objective package boundary.

Input And Output Flow

  • Config selects an objective id and metric.
  • Objective compile validates direction and optional evaluator binding.
  • Training/tuning uses that metric to choose best epochs or trials.

Common Confusing Concepts

Direction

minimize means lower is better, such as loss. maximize means higher is better, such as profit or F1.

Validation objective

Uses metrics from held-out validation samples during training.

Evaluation objective

Runs an evaluator and optimizes the evaluator metric, so it must match the selected evaluator id.
